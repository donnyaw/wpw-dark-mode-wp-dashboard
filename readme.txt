Refork of the original Simple Dark Mode for WP Dashboard plugin, maintained by donnyaw.

== Description ==

Just activate and get a dark WP-Admin Dashboard theme with a toggle in the top-right admin area.

This plugin is a refork of the original work by Roni Laukkarinen:
https://github.com/ronilaukkarinen/simple-dark-mode-for-wp-dashboard

== Installation ==

1. Upload the plugin package to the plugins directory.
2. Activate the plugin.
3. Done! Enjoy the dark!
